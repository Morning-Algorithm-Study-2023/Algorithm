
### 코드

```java
import java.util.*;

class Solution {
    public int solution(int distance, int[] rocks, int n) {
        int answer = 0;
        // 기준!
        int left = 0;
        int right = distance;
        int mid = 0;
        
        Arrays.sort(rocks);
        while (left <= right) {
            mid = (left + right) / 2; // 돌을 삭제할 범위
            int removeNum = 0;
            int prev = 0;
            
            // 1. 기준 범위(mid, 거리의 최솟값)보다 작을 때면 돌 삭제
            for (int i = 0; i < rocks.length; i++) {
                if (rocks[i] - prev < mid) { // 이 사이에 돌이 있으면 안되므로!
                    removeNum ++;
                } else {
                    prev = rocks[i];
                }
            }
            // 마지막 영역 체크
            if (distance - rocks[rocks.length-1] < mid) {
                removeNum++;
            }
            
            // 2. 삭제한 돌의 갯수에 따라 영역 변경해주기!
            if (removeNum <= n) { // 돌을 너무 조금 삭제함 -> 간격을 넓혀야함 -> 하한선 늘리기!
                answer = mid;
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        
        return answer;
    }
}
```
