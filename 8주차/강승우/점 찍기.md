## 코드
```java
class Solution {
    public long solution(int k, int d) {
        long answer = 0;
        for(int i=0; i<=d; i++){
            // k배수 이면
            if(i%k==0) {
                // y축 최대 길이 구하고
                long temp=(long)Math.sqrt(Math.pow(d,2)-Math.pow(i,2));
                // k배수 만큼만 더하기
                answer+=(temp/k)+1;
            }
        }
        return answer;
    }
}
```
