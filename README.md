## 계획 📆
#### 1. 일정
주 5일(평일), 오후 9시    
</br>

#### 2. 방식
* 오후 9시 코드 리뷰 시간 전까지 주어진 문제를 해결한 후 해당 코드를 업로드
* 소요 시간, 메모리, 풀이 방법 함께 기재
</br>

#### 3. 문제 선정
[프로그래머스](https://programmers.co.kr/)    

```
* 1주차 : 자료구조 - 힙/스택, 큐/해시 (Map, Set)
* 2주차 : 이분탐색/정렬
* 3주차 : 그리디
* 4주차 : 그래프
* 5주차 : 완전탐색
* 6주차 : BFS/DFS
* 7주차 : 시뮬레이션
```
</br>
